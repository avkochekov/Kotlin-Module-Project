# Консольное приложение для заметок
Учебный проект курса "Android-разработчик" Яндекс Практикума

### Программа позволяет 
 * создавать и удалять архивы с заметками, 
 * создавать и удалять заметки 
 * изменять текст заметок
 * просматривать список архивов/заметок
